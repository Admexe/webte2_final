Instructions for Roles

  • Admin:
    • Create and Manage Questions: Admins can create new voting questions and manage them for all users.
    • User Management: Admins can manage user accounts, including creating new accounts, editing existing ones, and deleting accounts. They can also change user roles.
    • Create Questions on Behalf of Other Users: When creating a new question, admins can specify on whose behalf the question is being created.
    • View and Analyze Results: Admins can view and analyze current voting results.
  • User:
    • Registration and Login: Users can register and log in to create and manage their own voting questions.
    • Create Questions: Users can create voting questions, choose question types (multiple choice or open-ended), and generate QR codes and unique codes for accessing the questions.
    • Participate in Voting: Users can participate in active votes and view results.
  • Guest:
    • Participate in Voting: Guests can participate in voting by entering a code on the homepage or scanning a QR code.
    • View Results: After voting, guests can view the voting results in graphical form.
    • No Registration Required: Guests can participate in voting and view results without the need to register.
